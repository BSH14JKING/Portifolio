E-commerce Website

This fully responsive e-commerce website was built to provide a seamless shopping experience across all devices. The project includes product listings with filtering options, a shopping cart with real-time updates, and a secure checkout process.

Features

- Responsive design that works on mobile, tablet, and desktop
- Product catalog with categories and search functionality
- Shopping cart with quantity adjustments and total calculation
- User authentication system
- Secure payment integration

Technologies Used

HTML5 CSS3 JavaScript React Node.js MongoDB

Challenges & Solutions

The main challenge was implementing the real-time cart functionality without page refreshes. I solved this by using React's state management to track cart items and localStorage to persist the cart between sessions.
